Version publique de production de l'application PoinçonLab, développée par Mehdi Benmouloud, Yacine Benosmane, Malcom Carlet, Justin Emanuel et Georgios Stephanou, étudiants du master MIASHS de l'Université Paul Valéry de Montpellier, durant l'année 2024-2025.

Cette application est destinée à faciliter l'entraînement de modèle ResNet34 pour détecter des motifs de poinçons sur céramique.

Elle génère un modèle .pt, qui peut ensuite être utilisé sur une interface graphique, comme par exemple  [l'application Crop Search DSP](https://github.com/The-dEnd/Crop-Search-DSP).
